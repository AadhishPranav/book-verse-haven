# Book Verse Haven

Book Verse Haven is an e-book library management system built using React for the frontend and Node.js with MongoDB for the backend. The platform allows users to browse, view, and manage e-books in a sleek, easy-to-use interface.

## Features

- **Browse Books**: A section to view all the available e-books.
- **Book Carousel**: A dynamic carousel showcasing popular books.
- **User Authentication**: User login and registration with form validation.
- **PDF Preview and Bookmarking**: Users can preview e-books and bookmark them for later.
- **Admin Interface**: Admins can manage e-books and users through a secure interface.
- **Backend Integration**: The backend handles all CRUD operations related to books, user data, and authentication.

## Tech Stack

- **Frontend**: 
  - React
  - Bootstrap
  - React Router

- **Backend**: 
  - Node.js
  - Express
  - MongoDB
  - Mongoose

## Setup and Installation

### Prerequisites

- Node.js installed on your machine
- MongoDB running locally or use MongoDB Atlas for a cloud database
- Git for version control

### Clone the repository

1. Clone the repository and navigate to the project directory.

### Frontend Setup

1. Navigate to the frontend directory.
2. Install the required dependencies.
3. Start the frontend server.

### Backend Setup

1. Navigate to the backend directory.
2. Install the required dependencies.
3. Create a `.env` file in the `backend` directory and add the MongoDB URI:

MONGO_URI=mongodb://localhost:27017/books-db


4. Start the backend server.

### Environment Variables

- `MONGO_URI`: The MongoDB connection string for your local or cloud database.

## Development

- Make sure both the frontend and backend servers are running during development.
- The frontend will be accessible at `http://localhost:3000` and the backend at `http://localhost:5000`.
